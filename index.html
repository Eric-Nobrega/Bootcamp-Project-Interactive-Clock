Bootcamp 2021: Interactive Clock

This is my Java project created in processing after 2 weeks of bootcamp here at City. I decided on this project because it would allow me to work on topics such as creating and utilising classes correctly, interactivty depending on factors such as the user's mouse position and code readability. In this project, the user is able to view the time in 3 different locations, and there is a sun/moon model that will tick with the current second, the sun and moon model will appear depending on the time.
